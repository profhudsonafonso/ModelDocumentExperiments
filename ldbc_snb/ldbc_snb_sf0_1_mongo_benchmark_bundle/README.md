# LDBC SNB SF0.1 MongoDB Benchmark Bundle

## 1. Purpose

This bundle contains the artifacts generated by the methodology pipeline for the LDBC SNB dataset.

It includes:

- final analytical matrix;
- G0-G9 activation matrix;
- MongoDB candidate specifications;
- full benchmark execution plan;
- smoke benchmark execution plan;
- validation files.

This bundle is intended to be copied to the benchmark server.

---

## 2. Dataset

Dataset: LDBC SNB Interactive v1
Scale factor: SF0.1
CSV serializer used in the notebook: CsvMergeForeign StringDateFormatter

The raw CSV files are not included in this bundle.

Expected dataset folder on the benchmark server:

```bash
/path/to/ldbc_snb/data/sf0.1
```

The folder should contain:

## 4. Recommended execution order

First run the smoke benchmark:

```bash
python run_ldbc_snb_mongo_benchmark.py \
  --data-dir /path/to/ldbc_snb/data/sf0.1 \
  --artifacts-dir /path/to/ldbc_snb_sf0_1_mongo_benchmark_bundle/mongo_configurations \
  --results-dir /path/to/results/ldbc_snb_sf0_1_smoke \
  --execution-plan benchmark_execution_plan_smoke.csv \
  --scale-label sf0.1 \
  --mongo-host 127.0.0.1 \
  --mongo-port 27017 \
  --batch-size 10000 \
  --repetitions 2 \
  --force-rebuild-db \
  --verbose
```

After validating the smoke run, run the full benchmark:

```bash
python run_ldbc_snb_mongo_benchmark.py \
  --data-dir /path/to/ldbc_snb/data/sf0.1 \
  --artifacts-dir /path/to/ldbc_snb_sf0_1_mongo_benchmark_bundle/mongo_configurations \
  --results-dir /path/to/results/ldbc_snb_sf0_1_full \
  --execution-plan benchmark_execution_plan.csv \
  --scale-label sf0.1 \
  --mongo-host 127.0.0.1 \
  --mongo-port 27017 \
  --batch-size 10000 \
  --repetitions 20 \
  --force-rebuild-db \
  --verbose
```

## 5. Benchmark groups

The execution plan uses three benchmark groups:

```text
primary
secondary_affected
control
```

Meaning:

- `primary`: main candidates selected by the activation logic;
- `secondary_affected`: alternative candidates activated by additional semantic families;
- `control`: baseline comparison candidates.
